## 介绍

**【中书君】** 是一款针对于 Rime 输入法引擎的管理器，最初的愿望是希望它能支持五笔输入方案的属性、词库等方面的管理。最终还希望它能完美地兼容更多输入方案（如简拼、双拼等）的管理，以提升 Rime 的使用体验。

目前，本软件还处于研发阶段，相应功能会持续更新，如有特殊需求，可提交 ISSUES.

如果对您有帮助，请帮我在右上角点个星，谢啦~~~ ⭐
